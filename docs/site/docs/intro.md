# 欢迎使用区块链实验室网站后端API

欢迎使用区块链实验室网站后端API文档。本文档提供了完整的API接口说明，帮助开发者快速集成和使用我们的服务。

## 📖 文档概览

本API文档包含以下内容：

- **快速开始**: 了解如何快速开始使用API
- **API接口**: 详细的接口说明，包括请求参数、响应格式等
- **参考文档**: 错误码、枚举值等参考信息

## 🚀 主要特性

- RESTful API设计
- Session认证机制
- 完整的CRUD操作
- 分页查询支持
- 数据统计功能
- 文件上传下载

## 📋 接口分类

本API包含以下主要模块：

| 模块 | 说明 |
|------|------|
| 认证管理 | 管理员登录、登出等操作 |
| 实验室成员管理 | 成员信息的增删改查 |
| 项目管理 | 项目信息的增删改查 |
| 博客文章管理 | 文章的发布、评论等 |
| 例会管理 | 例会信息的管理 |
| 文件管理 | 文件的上传下载 |
| 数据统计 | 系统数据的统计分析 |

## 🔒 认证说明

本API使用Session认证机制。管理员登录后，系统会返回一个Session ID，后续请求需要在Cookie中携带此Session ID。
